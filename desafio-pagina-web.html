Página Web Utilizando as Tags Aprendidas

A ideia deste Desafio de Projeto é elaborar um website, utilizando uma estrutura bem básica, utilizando tags aprendidas durante as aulas práticas:

  • Utilizar todas as tags explicadas nas aulas: <'h1> até <'h6>, <'p>, <'mark>, <'small>, <'i>, <'u>, <'strong>,
      , <'ul>, <'li>, <'a>, <'hr>, <'sub>, <'sup>, <'blockquote>;
    1. Utilizar novas tags sugeridas: <'font>, <'del>, <'p>, <'abbr> (a ideia é buscar estas tags na internet, entender como ela funciona e utilizar no texto).

  1. Historia da Web

    • Primeira Geração de Computadores:

      • Voltar

      - Abrangeu aproximadamente a década de 1940 até o final dos anos 1950.
      - Foi marcada principalmente pelo uso de componentes eletromecânicos e eletrônicos para realizar calculos científicos complexos durante a Segunda Guerra Mundial.
      - Harvard Mark I (1944); Colossus (1946); ENIAC (1946).
      - Surgimento termo "Bug" O termo "bug" surgiu para descrever problemas mecânicos ou técnicos causados pela presença de insetos, como mariposas, que eram atraídos pelas válvulas e relés dos primeiros computadores eletromecânicos.

    • Segunda Geração de Computadores:

      • Voltar

      - Abrangeu aproximadamente de 1959 até o final dos anos 1965.
      - Foi marcada pela substituição dos componentes eletromecânicos(valvulas), pelos transistores, isso resultou em computadores mais eficientes e compactos.

    • Terceira Geração de Computadores:

      • Voltar

      - Abrangeu aproximadamente de 1965 a 1970.
      - Foi marcada pela criação dos circuitos integrados: transistor, capacitores e resistores.

    • Quarta Geração de Computadores:

      • Voltar

      - Abrange a parti de 1971 em diante.
      - Criação do mouse pela Apple.
